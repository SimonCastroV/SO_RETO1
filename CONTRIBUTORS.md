## 📌 Descripción del Proyecto

Este proyecto consiste en la implementación de una shell educativa en lenguaje C, desarrollada como parte del curso de Sistemas Operativos. Incluye comandos básicos y avanzados para la gestión de archivos y operaciones del sistema.

## 🛠️ Cómo contribuir

Si deseas contribuir al proyecto:

1. Realiza un fork del repositorio.
2. Implementa nuevas funcionalidades o mejoras.
3. Envía un Pull Request para revisión.
